# Внесение в реестр - vnesenie-v-reestr.ru

Сайт для оказания услуг по внесению предприятий в различные реестры и сертификации.

## Описание проекта

Сайт предоставляет услуги по:
- Внесению в реестр Минпромторга
- Сертификации продукции
- Регистрации в Роскомнадзоре
- Другим видам сертификации и лицензирования

## Структура проекта

- `admin/` - Административная панель
- `certification/` - Страницы сертификации
- `config/` - Конфигурационные файлы
- `models/` - Модели данных
- `news/` - Новости и статьи
- `telegram_bot/` - Telegram бот для публикации новостей
- `uploads/` - Загруженные файлы
- `vkljuchenie-v-reestr-minpromtorga/` - Специальные страницы для Минпромторга

## Технологии

- PHP
- HTML/CSS/JavaScript
- MySQL (если используется)
- Telegram Bot API

## Установка и запуск

1. Склонировать репозиторий
2. Настроить конфигурацию в `config/`
3. Запустить веб-сервер

## Разработка

Для работы с проектом:
